# 3D Football
A 3D football game made in Unity

### Unity Version
2021.3.7f1

### Controls
`mouse` : Camera movement \
:arrow_left: or `a` : Move left \
:arrow_right: or `d` : Move right \
:arrow_up: or `w` : Move forward \
:arrow_down: or `s` : Move backward \
`space` : Dribble \
`rshift` : Shoot \
`tab` : Reset ball \
`esc` : Quit game
